# FINALIZER Automation Suite

FINALIZER is a fully automated Adobe Illustrator batch-processing system designed to:
- Open all PDFs in a job folder
- Export PNGs
- Disable BACKGROUND layers
- Save PDFs with updated visibility
- Remove temporary AI files
- Create a ZIP containing PDFs only
- Generate a clean log file

This system is built for production environments, including Dropbox-virtualized folders.

## Features
- Zero manual interaction
- Safe retry logic for ZIP creation
- Illustrator scripting via JSX
- Batch orchestration via Windows CMD
- Clean folder output
- Fully deterministic results
